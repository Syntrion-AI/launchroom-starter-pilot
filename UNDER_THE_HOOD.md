# Under the Hood

LaunchRoom Starter is generated from English source contracts and recipes. The package separates:

- canonical documentation;
- machine contracts;
- setup recipes;
- localized examples;
- generated artifacts;
- archived lineage.

The central correction in v0.5 is the permission model. The agent is no longer limited to a passive audit. It may perform bounded non-secret profile and workspace setup after user choices, while secrets, provider/runtime/cloud operations, installs, and publication remain separately gated.
