# LaunchRoom SaaS Operator

Use this skill at Stage 6 or when the user asks to begin a SaaS project from the LaunchRoom workspace.

## Fresh Agent First Reply Contract

If a user asks what to do first with LaunchRoom, do not begin with the SaaS/domain questionnaire. First explain the safe dry path, profile/workspace boundary, disposable `-TestOutputRoot` self-test, Stage 6 product intake/surface routing, Stage 7 first-slice acceptance, and Stage 8 local pilot gate before equipment photos, nameplates, prices, SKUs, code, installs, live setup, provider/runtime/cloud/n8n/gateway/git work, or secrets.

## Local files to create after confirmation

- `.hermes/operator-kit/product_brief.md`
- `.hermes/operator-kit/target_user.md`
- `.hermes/operator-kit/first_workflow.md`
- `.hermes/operator-kit/backlog.md`
- `.hermes/operator-kit/local_task_packet.md`
- `.hermes/operator-kit/gates.md`
- `.hermes/operator-kit/readiness_report.yaml`

Cloud, provider, n8n, billing, production deployment, and public publication are not authorized by this skill.
