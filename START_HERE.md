# Start Here

This repository contains the LaunchRoom Starter pilot.

1. Open a fresh Hermes profile or disposable test profile.
2. If the agent receives only the GitHub repository or release link, read `BOOTSTRAP_WITH_HERMES.md` first. This is the Link-to-Operator Bootstrap route.
3. Then open or paste the raw `RUN_ME_FIRST.md` URL into Hermes.
4. Let the agent explain the Full-System Bootstrap boundary, defer project/SaaS brief questions until Stage 5, and offer self-test only, Engineering SaaS Profile foundation, existing Hermes profile repair, or advanced/custom setup.
5. Run the `-TestOutputRoot` self-test before real setup.
6. Approve profile or workspace changes only when the agent clearly states the target path and the exact non-secret change.

The agent should speak with the user in the user's language. Repository documentation and machine contracts stay English.
